## 这是什么

**墨境** — AI 网文写作工作台。把原本需要手动粘贴的 225KB 提示词规则变成可视化的网站操作。

## 怎么打开

```bash
cd D:\建网站\mojing-app
pnpm dev
```
浏览器访问：
- 首页：http://localhost:3000
- 编辑器：http://localhost:3000/editor/demo-1
- 后台：http://localhost:3000/admin
- 项目全景面板：http://localhost:8899

## 技术栈

Next.js 16 + React 19 + TypeScript + Tailwind CSS 4 + TipTap 编辑器 + pnpm

## 项目结构

```
D:\建网站\mojing-app\          ← 网站代码
  src/app/editor/[id]/         ← 编辑器页面
  src/app/admin/               ← 后台管理系统
  src/lib/prompts/             ← AI 提示词系统
  src/components/              ← 通用组件
  src/lib/store.ts             ← 数据层（localStorage）

D:\建网站\mojing-docs\         ← 项目文档
  项目资料库/                   ← 原始素材、提示词模板、参考资料
  output/                      ← 各角色产出报告
  tasks/                       ← 当前任务文件
  screenshots/                 ← Playwright 自动截图
```
