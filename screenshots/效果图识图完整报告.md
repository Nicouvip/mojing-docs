## 全局设计语言总结

| 维度 | 规范 |
|------|------|
| **品牌色** | 绿色系 `#4a8e4a` ~ `#6b8c6e`（苔绿/深绿渐变） |
| **背景** | 浅米色 `#f9f7f5` ~ `#fafaf9` |
| **文字** | 黑 `#333` + 灰 `#999` |
| **卡片** | 圆角矩形 + 轻微阴影 + 白底/米底 |
| **按钮** | 绿色圆角 CTA + 灰色次按钮 |
| **字体** | 无衬线现代字体 + 手写体点缀 |
| **核心隐喻** | 纸张/书桌/书本 — 文学创作氛围 |
| **布局模式** | 三栏编辑器 / 卡片网格 / 左右分栏 |
