# 蛙蛙写作 UI 风格 → 墨境参考

## 全局设计语言
- **品牌色**: 薄荷翡翠绿 (#10B981)，浅色 #ECFDF5，深色 #059669
- **辅助色**: 蓝 #3B82F6、粉 #EC4899、橙 #F97316，各自搭配浅色背景
- **背景**: 页面色 #F9FAFB（浅灰白），侧栏/卡片纯白
- **文字**: 主 #111827、次 #6B7280、弱 #9CA3AF、占位 #D1D5DB
- **边框**: 极浅 #F3F4F6、正常 #E5E7EB
- **圆角**: sm 6px / md 8px / lg 12px / full 9999px
- **阴影**: 轻 0 1px 2px / 中 0 4px 12px / 重 0 10px 25px
- **字体**: -apple-system, PingFang SC, Microsoft YaHei, Noto Sans SC

## 布局结构
- **全局**: 左侧 220px 固定侧栏 + 右侧主区域（flex 布局）
- **侧栏**: Logo区(56px) + 导航分组 + 底部对齐的关于
- **顶栏**: 56px 固定，活动标签 + 余额 + VIP + 通知 + 头像
- **页面**: max-width 1280px 居中，padding 24px 32px

## 三页结构

### 页1: 首页工作台
- 一句话生成区（大卡片，中心标题 "一句话生成**爆款小说**"）
- 筛选标签行（题材/读者/视角/篇幅→胶囊标签）
- 输入框 + 生成按钮
- 左侧统计卡片（字数/使用次数/天数/作品数）
- 右侧四个入口大卡（小说/剧本/漫剧/投稿）
- AI工具广场（标签筛选 + 工具卡片网格 2列）

### 页2: 小说写作
- 大标题 "小说写作"
- 新的创作：三列入口卡（长篇/短篇/拆书）
- 近期作品：作品卡片（封面小图 + 标题 + 标签 + 元数据 + 操作按钮）
- AI工具广场：同上

### 页3: 三栏编辑器
- 左栏 260px: 作品信息/正文/草稿 tab，设定分组列表
- 中栏 flex: AI操作浮动条 + 工具栏 + 编辑区(padding 60px 120px)
- 右栏 320px: AI对话面板(消息 + 输入框)

## 组件库
- 按钮: btn-primary (绿色实心) / btn-secondary (白色描边)
- 胶囊标签: tag-capsule (白底灰边，active变浅绿底)
- 入口卡: entry-card (四色: 绿/蓝/粉/橙，圆角+hover上移阴影)
- 工具卡: tool-card (白底，头像+标题+徽章+描述+标签+统计)
- 作品卡: work-card (80x100封面 + 信息 + 操作)
- Tab下划线: tab-line (底部边框 + 激活下划线)
- 统计卡片: stats-card (大字 + 迷你图表 + 指标网格)

## 交互细节
- hover 微微上移 translateY(-2px) + shadow
- 边框聚焦变绿 + 3px 浅绿光晕
- 导航激活项: 浅绿背景 + 深绿文字
- 平滑过渡 0.15-0.2s ease

**Why:** 用户明确对标蛙蛙写作的 UI 排版风格，作为墨境前端的设计参考。
**How to apply:** 在构建墨境前端页面时，采用相同的设计令牌体系、布局结构和组件风格，但保留墨境的品牌色和功能差异。
