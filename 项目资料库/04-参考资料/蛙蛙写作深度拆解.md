# 蛙蛙写作深度参考资料（第二批）

用户提供了第二批蛙蛙写作深度拆解资料，补充了前一批缺少的细节。核心内容：

## 1. 前端工程化
- **三栏编辑器性能优化** — 拖拽调宽(rAF+pointer-events)、收起展开(transform 替代 width)、虚拟滚动、内存泄漏治理
- **重排隔离方案** — absolute定位脱离文档流、CSS变量统一管理宽度、BFC触发、合成层隔离
- **蛙蛙UI完整拆解** — 配色(#27C997薄荷绿)、四层布局(导航/顶栏/内容/辅助面板)、交互细节(空格唤起AI、选中快捷菜单)
- **Vue3三栏编辑器组件** — 完整代码，含拖拽调宽、Tab切换、AI面板、保存状态

## 2. 中间能力层
- **结构化内容引擎** — 自然语言↔结构化数据双向转换，37类网文专业字段
- **分层记忆引擎** — 长期记忆(结构化词条库+MySQL+向量库)、短期记忆(滚动重写固定长度)、计划记忆(四维:剧情/节奏/情绪/细节)
- **多Agent调度** — 策划/行文/校验/润色 四个Agent协同
- **个性化风格引擎(LPA)** — 文风画像+人物语料库+行为反馈闭环

## 3. 文风与低AI感
- **文风画像标准字段** — 叙事架构/句式语言/描写体系/用词规范/整体基调 5大模块
- **人物语料库标准字段** — 语气特征/表达习惯/用词偏好/情绪差异化/参考语料
- **低AI感三步循环** — Draft(初稿) → Polish(打磨) → Verify(校验)
- **可控随机扰动** — 温度动态波动、低频词替换、自然缺陷注入

## 4. RecurrentGPT 技术
- 四大记忆块: Output/Plan/Short-term Memory/Long-term Memory
- 与LSTM的对应关系
- 轻量复刻方案(基于第三方大模型)
- 结构化词条库替代纯文本向量检索

## 5. 模型底座(仅供参考)
- Weaver 1.8B/6B/14B/34B 多参数矩阵
- 垂域微调 + 指令回译 + Constitutional DPO
- 路由Agent按任务复杂度调度
- 小团队无需自研模型

**与墨境的融合点**: 分层记忆可对应墨境的全局摘要+L2上下文注入；文风画像可对应L4偏好学习；低AI感三步循环可整合进合规检测引擎。
