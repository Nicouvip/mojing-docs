# 墨境项目完整现状（全量备份分析）

## 项目定位
**墨境** — 沉浸式小说写作工具，极简优雅风格，对标蛙蛙写作的布局但有自己的品牌调性。

## 设计系统
- **主色**: #0071E3（苹果蓝）
- **强调色**: #27C997（绿色）
- **渐变**: linear-gradient(135deg, #667eea 0%, #764ba2 100%) — 紫靛蓝渐变
- **字体**: 系统默认 + Noto Serif SC（宋体/衬线体）
- **4套主题**: 默认（浅色）/ 暗夜（深蓝）/ 暖光（暖棕）/ 冷光（冷蓝）
- **圆角**: radius 0.75rem，shadcn/ui 体系
- **阴影**: 三级 soft/card/elevated

## 目录结构

### 前端（src/）
- `src/app/page.tsx` → WritingApp 组件（主入口，三栏布局）
- `src/app/editor/page.tsx` → UI 设计编辑器（407行）
- `src/app/canvas/page.tsx` → 画布页面
- `src/app/profile/page.tsx` → 个人资料
- `src/app/layout.tsx` → 根布局（AuthProvider + Toaster）
- `src/components/writing-app.tsx` → 主应用壳（侧栏+编辑区+右面板）
- `src/components/sidebar.tsx` → 左侧作品/章节树（260px）
- `src/components/editor.tsx` → 编辑器壳（含合规检测、偏好展示）
- `src/components/writing-editor.tsx` → TipTap 写作编辑器
- `src/components/right-panel.tsx` → 右侧AI面板（续写/AI工具/素材）
- `src/components/top-bar.tsx` → 顶栏（主题切换/用户/设置/字号）
- `src/components/ai-quick-actions.tsx` → AI快捷操作浮层
- `src/components/compliance-badge.tsx` / `compliance-actions.tsx` → 合规检测
- `src/components/cooling-panel.tsx` → 技法冷却面板
- `src/components/diff-viewer.tsx` → 差异查看器（偏好学习用）
- `src/components/material-visual-panel.tsx` → 素材可视化面板
- `src/components/creation-guide.tsx` → 新手引导
- `src/components/brain-jet-wizard.tsx` / `storm-wizard.tsx` / `incubate-wizard.tsx` / `title-wizard.tsx` → 四步流水线向导
- `src/hooks/` → use-auth, use-writing-store, use-generation, use-keyboard-shortcuts
- `src/lib/prompts/` → 完整提示词系统（含 compliance 合规检测引擎）
- `src/storage/` → Drizzle ORM schema + Supabase 客户端

## 当前问题（越改越乱的根源）
1. **版本碎片**: `src/components-v2/`、`src/v2/`、`page.tsx.bak` 等旧版本残留
2. **两套前端思路**: 新版 WritingApp（SPA式） vs 旧版 page.tsx.bak（React.createElement 手写）
3. **两套存储**: server/data/ JSON文件 vs Drizzle ORM（PostgreSQL/Supabase）
4. **配置不完整**: 部分配置文件可能缺失或错位

## 与蛙蛙写作的差异
- 蛙蛙是薄荷绿 + 暖感卡哇伊风格
- 墨境是苹果蓝 + 极简优雅风格 + 紫渐变
- 墨境有技法冷却、合规检测、偏好学习等系统级功能
- 墨境用 Noto Serif SC 宋体作为衬线字体选项

**Why:** 了解项目全貌才能有针对性地修复和迭代。
**How to apply:** 等用户确认方向后，先清理版本碎片，统一前端架构，修复配置错误让项目跑起来。
