# 知识库_叙事结构与角色工具


## 工具V：震撼开场引擎（前三章自动启用）

**三步结构**：
1. **角色困境句**：明确主语、关系和当前矛盾
2. **震荡细节句**：画面感强烈、反常或细思极恐的细节
3. **悬念出口句**：必须继续读下去才能解答的问题

**表现技法（至少使用两种，并标注用了哪几种）**：
- 身份错位 / 极端处境 / 情绪反差 / 失序时间 / 有限倒计时 / 禁忌颠覆 / 具体细节

**禁忌颠覆合规缓冲**：当使用“禁忌颠覆”类内容时，自动附加风险等级标注及缓冲修改方案。风险等级判断标准：🟢低风险（颠覆日常认知，如“我家的猫会说话”）；🟡中风险（颠覆社会常识，如“警察是坏人”）；🔴高风险（涉及政治、宗教、暴力、色情等敏感内容）。高风险内容建议替换，不强制修改但必须提示。

**格式要求**：白描，不使用任何修辞手法；快节奏短句，一句一换行（句号处强制换行）；口语化、强情绪；100字左右，三句话讲完更佳。


## 工具W：极端人设四维设计（人物设定时自动调用）

**四维因果链**：
1. **身份**（socioeconomic status + 基本信息 + 过去/现在处境）
2. **性格**（与处境相关，但异于常人，有故事性）
3. **思维行为模式**（因为性格，所以遇事会怎样思考和行动）
4. **语言特征**（因为常做某些行为，所以说话总是某种特征）

每个维度用1-2句精准概括。附加一句“为什么这个极端设定是合理的”解释。


## 工具X：起承转合大纲模式（长篇可选）

**【起】** 占比10%：人物→困境→机会→目标
**【承】** 占比50%：阻碍→行动循环，冲突递进升级
**【转】** 占比10%：意外→选择→格局提升
**【合】** 占比30%：最终行动→困境解除→主题升华

输出要求：使用“回顾已写完故事”视角，以具体事件形式给出，事件环环相扣因果清晰。用主角第一人称概述性视角，惜字如金，无过渡句废话。大纲末尾标注各阶段预估占比。


## 工具Y：短篇极速模式七要素（短篇创作时自动调用）

**七要素采集**：灵感、赛道（言情/世情/悬疑/脑洞等）、受众（男频/女频）、热点主题（家暴/霸凌/复仇等具体标签）、爆款元素（甜宠/逆袭/追妻火葬场等）、爆梗情节、数量。

**避坑规则**：避免职场/投行/商战/警察/黑道/官场/未成年恋爱；避免虚构神秘设定；避免专业性过强设定（医疗/法律/金融）。检测到风险时自动提醒并给出替代方案。

**逻辑链论证**：按顺序输出社会逻辑/因果逻辑/行动逻辑三重论证，然后输出四要素（人物+极端困境+契机+唯一目标）。最后用日常语言重述。


## 工具Z：救猫咪·开场共情锚点技巧（10种）

1. **救一只猫**：主角对弱小者施以援手。
2. **被误解的善良**：主角做了好事却被误认为是坏事。
3. **专业结果展示**（仅限开场/救猫咪场景适用）：不展示技能推导过程，只展示技能造成的震撼结果（如：一刀斩断瀑布，而非讲解刀法心诀）。与粗糙原则兼容，不描写具体操作过程。
4. **自嘲式幽默**：主角对自己的窘境开涮，显得不卑不亢。
5. **未完成的告别**：主角心里有一个来不及说的再见。
6. **默默的守护**：主角在无人知晓处做一件温柔的事。
7. **不公平的惩罚**：主角承受了不该他承受的代价。
8. **代人受过**：主角替别人扛下责任。
9. **逆流而行**：所有人都在做一件事，主角做相反的正确的事。
10. **最初的梦想**：主角有一个纯真的、与现实反差极大的梦想。

## 工具U：深度感替代方法 D-1 至 D-10

| 编号 | 方法名称 | 说明 |
|------|----------|------|
| D-1 | 身体反应链 | 用一连串身体细微变化构成段落 |
| D-2 | 微环境锚定 | 将情感投射到极小的环境细节（灰尘/水渍等） |
| D-3 | 感官错位 | 用非对应的感官描写（如“听见颜色”）制造陌生化 |
| D-4 | 物心转换 | 通过物品的状态变化反映内心 |
| D-5 | 未完成动作 | 描写一个进行到一半被打断的动作，留下悬念 |
| D-6 | 时间拉伸 | 将极短的瞬间拉长，放大细节 |
| D-7 | 空间压缩 | 把大空间浓缩为几个关键细节快速掠过 |
| D-8 | 多线程感官 | 同时写2-3条感官线，制造沉浸感 |
| D-9 | 隐藏因果 | 只写现象不写原因，读者自己拼凑 |
| D-10 | 意象锚定 | 全章用一个核心意象串联 |
