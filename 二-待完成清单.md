# 二、待完成清单

## 1. AI 功能缺口
| 缺失项 | 严重度 | 说明 |
|--------|:------:|------|
| API Key 硬编码在源码中 | 🔴 | 4 个 route.ts 都含明文 key，有泄露风险 |
| 无流式输出（Streaming） | 🟡 | AI 结果需等全部返回才展示，体验差 |
| 无请求超时处理 | 🟡 | fetch 无 AbortController/signal |
| 无重试机制 | 🟢 | 网络失败只显示"请求失败"，无重试按钮 |
| 灵感爆裂/书名炼金术未实现 | 🟡 | 按钮存在但无实际功能 |

## 2. 编辑器缺口
| 缺失项 | 严重度 | 说明 |

| 无手动输入字数目标的入口 | 🟢 | 只有 `+500` 按钮 |
| 无快捷键提示 | 🟢 | 工具栏无快捷键标注 |
