**微任务：按钮点击反馈动效实现 —— 3 种状态、零新依赖**

1. 检查当前项目中所有 `<button>` 及可点击元素（Card / IconButton / Tag）的交互反馈
2. 用纯 CSS（transition + animation）实现三态动效：
   - 悬停态：轻微上浮 + 阴影加深（transform: translateY(-2px) + box-shadow 变化）
   - 按下态：缩放 0.96 + 背景色加深 10%（active 伪类，时长 100ms）
   - 成功态：完成后做一个 400ms 的绿光脉冲（border-color / box-shadow 动画，仅成功回掉时通过 class 触发）
3. 抽取为通用 mixin `@mixin btn-feedback` 放入 `src/styles/mixins.scss`，所有按钮统一样式
4. 输出到 `output/动效-按钮三态反馈方案.md`

验收标准：打开任意页面点击任意按钮，眼睛能看到三层反馈（浮起→按下→脉冲），F12 检查无新 npm 依赖。
