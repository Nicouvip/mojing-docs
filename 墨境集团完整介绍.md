## 一、我们做什么

**墨境（MoJing）**是一个AI驱动的长篇小说写作平台。官网 `mojing-app.vercel.app`。

**墨境集团**是管理这个项目的AI团队体系——31个AI角色各司其职，由"总管"调度，用户在"虚拟总部"中以游戏化方式管理。

**虚拟总部**是将集团管理游戏化的2D空间——像一个《模拟人生》风格的办公楼，用户以CEO角色走进去，看到所有AI员工在工作，可以点击他们查看任务、下命令、拉进会议室。

| 层 | 技术 |

| **游戏引擎** | Phaser.js 4（Canvas 2D渲染） |
| **UI叠加层** | React 19 + TypeScript |
| **实时通信** | Socket.IO（多人位置同步+聊天） |
| **服务端** | Node.js + Express（命令API） |
| **打包工具** | Vite 8 |
| **数据库** | 本地 SQLite → 可迁 Supabase |
| **素材格式** | PNG Spritesheet（精灵图） |

### 启动方式
```bash
双击 start.bat 或：
node server.js          # Socket.IO + API → :8802
node bot-runner.js      # AI角色入职
npx vite                # 前端 → :5173
```

## 七、设计需求

### 需要设计师产出

1. **角色spritesheet**（48x64px × 4方向×4帧 = 16帧）：
   - CEO角色（你）— 有明显辨识度
   - 各组组长 — 不同颜色/配饰区分
   - 普通组员 — 统一风格

2. **家具spritesheet**：
   - 办公桌、椅子、会议桌、盆栽、饮水机、沙发

3. **地板纹理**（40x40px可平铺）：
   - 木纹、地毯、瓷砖

4. **UI面板视觉设计**：
   - 侧栏（在线列表+聊天区）
   - 角色点击菜单（Popover）
   - 气泡样式
   - 斜杠命令面板

### 视觉风格参考

- 苔绿(`#6b8c6e`) + 米白(`#faf9f6`) 主色调
- 2D俯视视角，像《星露谷物语》
- 简约线条，不花哨
- 参考：墨境网站的主题风格

### 技术约束

- 所有素材必须是PNG格式
- 角色和家具用spritesheet（一图多帧）
- 地板纹理用40×40px可平铺
- 地图可用Tiled编辑器设计（免费开源）
- 不需要3D、不需要WebGL特效
