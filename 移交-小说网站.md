## 技术栈

- 前端：Next.js 16 + React 19 + Tailwind 4 + shadcn/ui
- 编辑器：TipTap 3
- 页面编辑器：GrapesJS（localhost:3000/page-editor?path=/）
- 数据库：Supabase + Drizzle ORM + localStorage fallback
- 认证：NextAuth.js 5 beta
- AI：DeepSeek V4 Flash

## 关键文件

- 主配置：`src/app/layout.tsx`、`package.json`
